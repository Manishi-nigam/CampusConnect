# CampusConnect

CampusConnect is a full-stack web application built using **Spring Boot** and **Angular**.

## Features
- Student management
- Student-wise event listing
- Google OAuth2 login
- REST APIs with Spring Boot
- Angular standalone components
- Bootstrap-based UI

## Tech Stack
**Backend**
- Java
- Spring Boot
- Spring Security (OAuth2)
- PostgreSQL
- JPA / Hibernate

**Frontend**
- Angular (Standalone Components)
- Bootstrap
- RxJS
- Angular Router

## Authentication
- Google OAuth2 (Spring Security)
